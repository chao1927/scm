## 2. 页面清单

| 页面 | 主要内容 |
| --- | --- |
| 库存工作台 | 缺货、预占失败、冻结、对账差异、调整待审 |
| 库存余额 | 仓库、货主、SKU、批次、库存状态、实物、可用、预占、冻结、在途 |
| 可用库存 | ATP 可用数量、预计入库、在途、可用等级 |
| 预占管理 | 预占单、来源系统、来源单、类型、请求数量、预占数量、状态 |
| 冻结解冻 | 冻结单、原因、SKU、冻结数量、冻结状态、审批状态 |
| 库存调整 | 调整单、类型、调整数量、原因、调整状态、审批状态 |
| 库存流水 | 流水号、类型、方向、变化数量、前后可用、来源单 |
| 库存快照 | 快照号、日期、仓库、类型、SKU 数、实物总量、状态 |
| 库存对账 | 对账单、仓库、中央数、WMS 数、差异行、状态 |
| 事件日志 | 事件 ID、事件名称、来源系统、来源单、处理状态、失败原因 |
| 参数配置 | 可用口径、预占超时、库存维度、对账策略 |
| 操作日志 | 操作人、对象类型、对象单号、动作、结果、操作时间 |
| 枚举配置 | 枚举类型、枚举值、显示名称、排序、启停状态 |

## DDD 对齐说明

本文属于 **中央库存上下文**。设计时应把页面、字段和流程统一回到该上下文的模型边界，避免跨上下文直接修改数据。

| DDD 项 | 对齐口径 |
| --- | --- |
| 限界上下文 | 中央库存上下文 |
| 核心聚合 | InventoryAccount、InventoryReservation、InventoryLedger、InventoryAdjustment |
| 数据主权 | 可用库存、预占、释放、扣减、冻结和流水 |
| 生产事件 | 只发布本上下文已经发生的业务事实 |
| 消费事件 | 消费外部事实时必须记录 event_id、幂等键、处理状态和失败原因 |
| 查询模型 | 列表、看板、导出可使用读模型，不强行加载聚合 |
