# WMS系统产品功能设计

> 本文是 `docs/04-子系统功能设计/WMS系统` 的唯一产品功能设计文档，从产品视角描述菜单、页面、按钮、操作逻辑、角色权限、数据权限、查询、列表、详情、新增、修改、登录/登出和审计。领域规则参考 `docs/03-核心业务模型/03-WMS领域模型`，表字段参考 `docs/05-子系统数据库设计/03-WMS系统数据库设计.md`。

## 1. 产品定位与设计边界

| 项 | 设计 |
| --- | --- |
| 系统 | WMS系统 |
| 设计对象 | 后台管理页面、业务操作页面、配置页面、日志页面和权限控制 |
| 不承载内容 | 数据库 DDL、领域聚合细节、跨系统接口实现；这些分别在数据库设计、核心业务模型、接口事件文档维护 |
| 产品原则 | 先通过权限系统完成登录和授权，再按菜单进入功能；所有写操作必须校验按钮权限、数据权限、状态机和幂等键 |

## 2. 登录、登出与全局交互

| 页面/动作 | 使用角色 | 展示字段 | 操作逻辑 |
| --- | --- | --- | --- |
| 登录页 | 未登录用户 | 账号、密码、验证码/MFA、租户/组织可选 | 用户输入凭证后调用权限系统认证；成功写入 access_token/refresh_token，拉取菜单、按钮权限和数据范围；失败记录登录日志。 |
| 退出登录 | 已登录用户 | 当前用户、当前系统、会话过期时间 | 用户点击退出后吊销当前 Token，清理前端缓存、菜单缓存和权限缓存，并跳转登录页。 |
| 个人中心 | 已登录用户 | 姓名、账号、手机号、邮箱、角色、所属组织 | 读取当前用户资料和角色权限；修改密码或个人资料时记录操作日志。 |

全局页面规则：

- 所有列表页默认包含查询、重置、分页、排序、导出入口；导出必须受数据权限约束。
- 所有详情页默认包含基础信息、明细信息、状态流转、领域事件、操作日志。
- 所有新增/编辑页默认校验必填、唯一、枚举、引用关系、乐观锁和状态机。
- 所有写操作默认记录操作日志；会改变业务事实的操作写入领域事件发布表。

## 3. 角色、菜单权限与数据权限

| 角色 | 可用功能 | 典型动作 | 数据范围 |
| --- | --- | --- | --- |
| 收货员 | 到货登记、收货 | 扫码、点数、登记差异 | 按仓库、货主、库区授权 |
| 质检员 | 质检 | 判定合格、不合格、冻结 | 按仓库、货主、库区授权 |
| 上架员 | 上架 | 按推荐库位上架 | 按仓库、货主、库区授权 |
| 拣货员 | 拣货、容器绑定 | 按拣货单从指定库区库位拿货，放入周转箱/播种格口 | 按仓库、货主、库区授权 |
| 复核/包装员 | 复核、打包、打单 | 扫描容器、复核商品、包装称重、贴面单 | 按仓库、货主、库区授权 |
| 发货员 | 交接 | 承运商交接、出库确认 | 按仓库、货主、库区授权 |
| 仓库主管 | 波次、任务、盘点 | 分配任务、处理异常 | 按仓库、货主、库区授权 |

数据权限维度：

| 维度 | 控制方式 | 页面效果 |
| --- | --- | --- |
| 组织 | 按当前组织、下级组织或自定义组织集合过滤 | 列表、详情、导出只返回可见组织数据 |
| 仓库 | WMS、库存、OMS、BMS 按仓库授权过滤 | 未授权仓库不可查询、不可操作 |
| 货主 | 多货主场景按货主授权过滤 | 只能查看和操作授权货主的单据和库存 |
| 供应商/客户 | 外部协同用户绑定供应商或客户 | 只展示自身协同单据，不展示其他主体数据 |
| 本人/部门 | 普通业务用户可限制为本人或本部门 | 请购、待办、审批等页面优先使用该范围 |

| 盘点管理页 | `/wms/count-plans` | 创建盘点计划、执行盘点、处理差异 | 新增、下发、复盘、确认差异 |
| 异常处理页 | `/wms/exceptions` | 处理短收、超收、错货、破损、复核差异 | 分派、处理、关闭 |
| 操作日志页 | `/wms/operation-logs` | 查询仓内关键操作 | 查询、导出 |
| 枚举配置页 | `/wms/enums` | 维护 WMS 页面枚举项 | 新增、编辑、排序、停用 |

## 5. 页面详细设计

### WMS 工作台

| 项 | 设计 |
| --- | --- |
| 菜单路由 | `/wms/workbench` |
| 主要用户 | 业务操作员、业务主管、系统管理员；外部协同页面按供应商/客户/货主数据范围过滤 |
| 业务对象 | 入库单 |
| 关联表/读模型 | `wms_inbound` |
| 页面目的 | 展示入库、出库、异常、待办任务 |
| 菜单权限 | `wms:workbench:read` |

| 区域 | 字段/按钮 | 规则 |
| --- | --- | --- |
| 查询条件 | WMS 入库单号、来源单号、采购、调拨、销售退货、其他、待到货、到货中、收货中、待质检、待上架、部分上架、已上架、已关闭、已取消 | 支持组合查询；未选条件默认按用户数据权限过滤 |
| 排序 | 创建时间、更新时间 | 默认按更新时间倒序；支持分页，每页 10/20/50 条 |
| 列表展示 | 待收货、待质检、待上架、待拣货、待复核、异常数、入库单ID、WMS 入库单号、来源单号、采购、调拨、销售退货、其他 | 列表展示业务识别字段、状态、关键金额/数量和更新时间 |
| 顶部按钮 | 查询、重置 | 根据按钮权限和数据范围显示 |
| 行内操作 | 查看详情、查看待办、进入处理 | 终态数据只允许查看、导出或复制，不允许直接编辑 |
| 详情展示 | 入库单ID、WMS 入库单号、来源单号、采购、调拨、销售退货、其他、待到货、到货中、收货中、待质检、待上架、部分上架、已上架、已关闭、已取消、预计到仓时间、到货时间 | 详情页展示基础信息、明细、状态流转、事件记录和操作日志 |
| 新增页面 | 入库单ID、WMS 入库单号、来源单号、采购、调拨、销售退货、其他、预计到仓时间、到货时间 | 新增时校验必填、唯一、枚举、引用关系和数据权限 |
| 修改页面 | 入库单ID、WMS 入库单号、来源单号、采购、调拨、销售退货、其他、预计到仓时间、到货时间 | 已生效/终态数据只允许走变更、关闭、停用或补偿流程 |

| 操作 | 权限点 | 操作逻辑 | 结果/审计 |
| --- | --- | --- | --- |
| 查询 | `wms:wms_workbench:read` | 读取读模型或列表投影，不修改业务状态 | 不记录业务事件，可记录访问日志 |
| 查看待办 | `wms:wms_workbench:page` | 执行业务动作，后端校验状态、权限、幂等键和参数 | 记录操作日志，按需要发布领域事件 |
| 进入处理 | `wms:wms_workbench:page` | 执行业务动作，后端校验状态、权限、幂等键和参数 | 记录操作日志，按需要发布领域事件 |

## 7. 关键操作逻辑

| 操作类型 | 前置校验 | 状态/数据变化 | 事件与审计 |
| --- | --- | --- | --- |
| 新增 | 菜单权限、按钮权限、必填、唯一、引用主数据有效 | 创建草稿或初始状态数据 | 记录操作日志；必要时发布已创建事件 |
| 修改 | 按钮权限、数据权限、乐观锁、非终态、字段可编辑 | 更新业务字段或生成变更记录 | 记录前后快照；关键变更发布事件 |
| 删除 | 通常只允许草稿或未引用数据 | 逻辑删除，历史单据不物理删除 | 记录删除原因和操作者 |
| 提交/发布/审批 | 状态机允许、审批权限、幂等键未重复 | 推进状态，锁定或发布业务事实 | 写领域事件表，刷新读模型 |
| 取消/关闭/停用 | 无未完成依赖或允许补偿路径 | 进入终态或禁用状态 | 记录原因；通知下游或生成待办 |
| 查询/导出 | 查询权限、数据范围、脱敏策略 | 不改变业务状态 | 导出记录审计日志 |
