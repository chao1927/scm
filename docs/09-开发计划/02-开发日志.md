# 供应链系统开发日志

## 使用规则

- 每完成一个可验收功能切片，立即在对应系统下新增一条记录；不得只在聊天中说明完成。
- 记录“新增、修改、删除、迁移、接口、事件、测试、已知问题”，保证其他模型可无缝接手。
- 不删除历史记录；修复通过新增日志关联原编号完成。
- 状态仅允许：`计划中`、`开发中`、`已完成`、`已回滚`、`阻塞`。
