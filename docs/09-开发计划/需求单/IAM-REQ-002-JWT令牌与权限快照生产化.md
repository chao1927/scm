# IAM-REQ-002 JWT 令牌与权限快照生产化

## 目标

把权限系统从首轮本地会话升级为可供业务系统调用的认证授权事实源，补齐 JWT 真签名、Token 校验、权限快照、数据范围解析、会话撤销和缓存失效事件。

## 范围

- 登录/刷新签发 HMAC-SHA256 JWT，保留会话状态作为撤销事实源。
- OpenAPI 支持 Token 校验、当前权限快照、权限检查和数据范围解析。
- 权限快照可重建、缓存、失效，并生产 `PermissionSnapshotInvalidated`。
- 相关应用服务、接口、迁移和测试。

## 验收

- 篡改 Token 或撤销会话后校验失败。
- 用户角色、角色权限和数据范围可合成为权限快照。
- 权限检查返回 allowed/denyReason，数据范围按角色合并。
- IAM 模块测试和全量测试通过。
