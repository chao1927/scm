# MDM-REQ-005 OpenAPI 快照与事件入口

## 目标

为采购、OMS、WMS、TMS、BMS 等系统提供稳定的主数据查询、批量校验和快照接口，并提供内部事件入口承接审批、回执、供应商资料变更等外部事实。

## 范围

- OpenAPI 查询：批量查询、批量校验、单条快照查询。
- 事件入口：幂等写入 Inbox，支持成功、忽略和失败状态。
- 发布回执：兼容既有发布回执服务。
- 数据质量联动：外部资料变更/承运服务确认异常可生成质量问题。

## 验收

- 批量校验默认要求主数据处于启用状态。
- 请求指定版本时，版本不匹配返回失败明细。
- 内部事件入口重复事件命中幂等，不重复产生副作用。
- 事件入口具备 Inbox 持久化、状态更新和测试覆盖。
