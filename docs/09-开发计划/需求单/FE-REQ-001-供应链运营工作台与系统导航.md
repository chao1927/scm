# FE-REQ-001 供应链运营工作台与系统导航

## 目标

在后端核心系统首轮接口基本完成后，补齐前端统一入口，让运营、采购、仓库、结算和技术运维能看到各系统完成状态、关键待办、集成风险和当前可操作页面。

## 范围

| 类型 | 内容 |
| --- | --- |
| 页面 | 供应链运营工作台、供应商 ASN 发货通知页 |
| 导航 | 供应商协同、采购、WMS、中央库存、OMS、TMS、BMS、主数据、权限、集成中心 |
| 工作台 | 系统完成状态、P1/P2/P3 未完成排序、关键指标、待办清单、集成风险 |
| 交互 | 侧边栏切换页面、刷新、查看待办、保留 ASN 新建/提交/取消操作 |
| 样式 | Ant Design 中后台风格，响应式布局，避免营销页和装饰性卡片堆砌 |

## 验收

- 首页是可用工作台，不是说明页或营销页。
- 已完成/未完成信息与开发日志排序一致。
- 侧边栏可以切换到 ASN 页面。
- 移动端和桌面端布局不发生文字重叠。
- 前端 `npm run build` 通过。
