# MDM-REQ-004 导入导出与数据质量治理

## 目标

补齐主数据批量维护能力，让主数据专员可以创建导入任务、记录校验错误、执行/取消任务、创建导出任务，并将发现的数据问题进入治理闭环。

## 范围

- 导入任务：创建、查询、校验、执行、取消、错误明细。
- 导出任务：按类型和字段创建异步导出任务。
- 数据质量问题：发现、分派、修复、验证、关闭。
- 事件：导入任务和质量问题关键状态写入 MDM Outbox。

## 验收

- 导入任务同 `typeCode + fileHash + importMode` 幂等返回既有任务。
- 有失败行时任务进入部分失败或失败状态，并保留错误明细。
- 数据质量问题状态迁移受聚合约束，不允许跳过分派/修复直接关闭。
- 应用服务、接口层和领域模型均有测试覆盖。
