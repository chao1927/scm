# WMS-REQ-006 波次释放与拣货任务

## 1. 背景与目标

出库单分配后，仓内执行需要通过波次组织拣货任务。本切片先完成波次创建/释放、拣货任务创建和 PDA 拣货扫描，形成 `WmsPickCompleted` 事件。

## 2. 范围

| 类型 | 内容 |
| --- | --- |
| 包含 | 波次创建、波次释放、拣货任务创建、PDA 拣货扫描、拣货完成事件 |
| 不包含 | 容器绑定、复核包装、路径推荐、短拣异常、波次自动生成策略 |

## 3. 接口

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| `POST` | `/api/wms/v1/waves` | 创建波次 |
| `POST` | `/api/wms/v1/waves/release` | 释放波次 |
| `POST` | `/api/wms/v1/pick-tasks` | 创建拣货任务 |
| `POST` | `/api/wms/v1/pda/pick-tasks/scan` | PDA 拣货扫描 |
