# WMS-REQ-007 容器绑定与复核包装

## 1. 背景与目标

拣货完成后，需要把拣货结果绑定到周转容器，并通过复核包装确认货品可进入发货交接。本切片完成容器绑定、封箱、复核包装创建和确认。

## 2. 范围

| 类型 | 内容 |
| --- | --- |
| 包含 | 容器绑定、容器封箱、复核包装创建、复核确认、Outbox 事件 |
| 不包含 | 面单打印、称重、包裹明细、短拣异常、TMS 运单联动 |

## 3. 接口

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| `POST` | `/api/wms/v1/containers/bind` | 绑定周转容器 |
| `POST` | `/api/wms/v1/containers/seal` | 容器封箱 |
| `POST` | `/api/wms/v1/packing` | 创建复核包装单 |
| `POST` | `/api/wms/v1/packing/verify` | 确认复核包装 |
